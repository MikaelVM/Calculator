# Project - Create a calculator
Development of a simple calculator.

# Requirements
## Priority 1 - The basic calculator
- Take 2 numbers as input.
- User should be able to chose 1 of the following operations: +, -, * and /.
- Provide the result of the operation.
## Priority 2 - Expanded Calculator
- Program must continue after providing the result to make further calculations.
- Program must have a specific command to terminate, like 'q' for quit.
## Priority 3 - Advanced Calculator
- Support operators for square root and power.
- Have input validation.
- Possibly make the code more modular and organized.
